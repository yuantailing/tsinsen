时间限制：1.0s 内存限制：512.0MB

【问题描述】

已知一个有理数类Zrf_Ratio，实现如下的操作符重载形式：

    friend zrf_Ratio operator-(const zrf_Ratio&);

    friend zrf_Ratio operator+(const zrf_Ratio&, const zrf_Ratio&);

    friend zrf_Ratio operator-(const zrf_Ratio&, const zrf_Ratio&);

    friend zrf_Ratio operator*(const zrf_Ratio&, const zrf_Ratio&);

    friend zrf_Ratio operator/(const zrf_Ratio&, const zrf_Ratio&);

 【测试】

输入为4个整数，输出见main函数。

【输入样例】

1 7 26 65

【输出样例】

zrf is:1/7; ssh is:2/5

-zrf is:-1/7(zrf-ssh) is:-9/35

(zrf+ssh) is:19/35

(zrf*ssh) is:2/35

(zrf/ssh) is:5/14 

 

编译语言：CPP
